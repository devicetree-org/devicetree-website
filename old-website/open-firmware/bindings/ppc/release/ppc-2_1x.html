Updates to PPC Binding in Version 2.1 (Approved Version)

  • Changed second paragraph of Section 8.4 into a note, as follows: 
    Note: There used to be a paragraph here that said an area of memory 
was to be reserved by the client program for the exclusive use of Open 
Firmware.  This requirement has been removed, since the sharing of 
interrupt vectors on these platforms has not been found to be practical.
